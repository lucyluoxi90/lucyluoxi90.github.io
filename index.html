Hexo for Beginner

The following is a simple guide with a list of instructions and corresponding codes that I used to install Hexo on my Mac.

Before installing Hexo, I downloaded Node.js and Git from their official websites listed below:
https://nodejs.org/en/
https://git-scm.com/downloads

After installing Node.js and Git on the computer, open up terminal to test whether the installations were successful. Input the lines below in terminal and they should return the versions for the two applications respectively to confirm they were installed. 

$ node -v
$ git version

Install Hexo

In the commandline enter:

$ sudo npm install -g hexo-cli

This will result in a prompt asking for permission from user. Enter the password in the prompt and then Hexo will be downloaded and installed. Once Hexo is installed, create your hexo project using the following line. Note: blog_name is the name you choose for your blog.

$ hexo init blog_name

Run the following command to start the server. http://localhost:4000 is the link to your website. 

$ hexo server

Now you could start using your new Hexo site by creating a new post on the blog by running the command below. The name of the post will be in quotation.

$ hexo new “post_name”

You can start blogging. Open the file post_name md. in blog_name source/_posts to write the content of your first post.
