Regular expressions

A regular expression or regex defines a pattern with descriptions and it is used to find a matching pattern in strings.

The pattern is written in raw string notation to avoid escape characters. The raw string starts with r‘’. For example r‘Hello/n’ does not represent a string with ‘Hello’ and a single new line, instead it represents the string literal of ‘Hello/n’ ending with a backslash followed by character n.

^Hello matches a string that begins with ‘Hello’
/n$ matches a string that ends with ‘/n’
* matches the preceding regex zero or more times
+ matches the preceding regex one or more times
? matches the preceding regex zero or one times
{n} matches the preceding regex with n numbers of repetitions
. matches any character except a newline
| matches with any of the expressions separated by pipe

[Hello] matches a string with any characters inside the brackets
[^Hello]matches a string with any characters that’s not in the the brackets
